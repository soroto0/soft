0 слов
Сгенерированный текст — черновик: вычитай и переработай перед озвучкой, иначе это «inauthentic content».
«Глубокий голос» — компрессия + нижние частоты + нормализация, как у диктора документального фильма (звучит плотно и убедительно).
Фоновая музыка (можно несколько треков — чередуются)
Несколько треков склеиваются в плейлист и зацикливаются под всю длину видео. Музыка должна быть лицензионной (YouTube Audio Library и т.п.). «Подобрать под жанр» берёт трек сам из библиотеки в Настройках — без выбора папки.
ASMR-звуки быта (эффект присутствия)
Софт сам раскидывает короткие звуки быта по таймлайну тихо под голосом — создаёт «честное» ощущение присутствия. Звуки скачай бесплатно на pixabay.com/sound-effects.
Таймкоды фраз — основа монтажа: по ним ставятся смены кадров и оверлеи.

Видео

«Свои фото/видео» копируются в проект и идут в рендер вперемешку со стоками (фото автоматически оживают зумом Ken Burns).
«Только стоки» почти не генерирует ИИ-кадры (ИИ подключается лишь если сток совсем ничего не нашёл — редкий случай). «Микс» намеренно генерирует заданную долю планов через ИИ равномерно по ролику — так видео не выглядит сплошным стоком, но и не тратит кредиты на каждый кадр. «ИИ в едином стиле» генерирует ВСЁ — медленно, зато максимально цельно.
Формат: timecode | тип | контент | позиция | длительность. Типы: popup, lower3, callout, counter, bars, timeline, compare, banner. Автопредложение — черновик: обязательно вычитай перед рендером.
Эффекты поверх кадра
.mp4 добавится сам
Смены кадров — по границам фраз; ритм и переходы — от seed проекта. Выход: output_final.mp4. Живой прогресс ffmpeg — в консоли внизу.

📦 Экспорт + Premiere

Premiere Pro: File → Import → sequence.xml — готовый таймлайн: клипы по таймкодам, озвучка под ними. Субтитры: File → Import → subs/voiceover.srt. Дальше — обычный монтаж: заменить неудачные планы, добавить переходы.
