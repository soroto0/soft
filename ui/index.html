🎙 Озвучка

«Глубокий голос» — компрессия + нижние частоты + нормализация, как у диктора документального фильма (звучит плотно и убедительно).
Фоновая музыка
Трек подкладывается под озвучку с автоприглушением. Музыка должна быть лицензионной (YouTube Audio Library и т.п.) — источник пишется в журнал.

💬 Транскрибация

Таймкоды фраз — основа монтажа: по ним ставятся смены кадров и оверлеи.

🎞 Раскадровка и стоки

«Свои фото/видео» копируются в проект и идут в рендер вперемешку со стоками (фото автоматически оживают зумом Ken Burns).
Материал подбирается под то, о чём говорится в каждый момент озвучки. Результат: storyboard/, timeline.json и sequence.xml для Premiere.

✨ Оверлеи (моушн-графика)

Формат: timecode | тип | контент | позиция | длительность. Типы: popup, lower3, callout, counter, bars, timeline. Автопредложение — черновик: обязательно вычитай перед рендером.

🎬 Авторендер

Эффекты поверх кадра
.mp4 добавится сам
Смены кадров — по границам фраз; ритм и переходы — от seed проекта. Выход: output_final.mp4. Живой прогресс ffmpeg — в Журнале и в статус-баре.

📦 Сборка + Premiere

Premiere Pro: File → Import → sequence.xml — готовый таймлайн: клипы по таймкодам, озвучка под ними. Субтитры: File → Import → subs/voiceover.srt. Дальше — обычный монтаж: заменить неудачные планы, добавить переходы.
